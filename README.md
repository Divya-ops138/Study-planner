# 📚 Study Planner – Panda Planner 🐼

A beautiful, responsive study planner web app made with HTML, CSS, and Bootstrap.

## 🌟 Features:
- 📅 Timetable page
- 📝 Study notes cards
- ✅ Task progress bar
- 🎠 Image carousel
- 💕 Custom footer with contact info
